# WibuOtaku

Scaffold Next.js untuk tracker anime, manga, dan light novel sesuai spesifikasi di [OTAKU-HUB-SPEC (1).md](OTAKU-HUB-SPEC%20(1).md).

## Jalankan

```bash
npm install
npm run dev
```

## Cakupan awal

- Dashboard editorial bergaya premium, bukan layout generik.
- Route inti sesuai sitemap PRD.
- Data mock yang siap diganti ke API, database, dan scraper.
- Fondasi PWA-ready lewat manifest App Router.
