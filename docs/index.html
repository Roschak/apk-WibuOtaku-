Cara Install di HP Android

Hanya 4 langkah sederhana. APK bisa langsung didownload tanpa Play Store.

1

Download APK

Klik tombol Download di atas, atau buka halaman Releases di GitHub.

2

Buka File

Buka file `.apk` yang sudah didownload di folder Downloads HP kamu.

3

Izinkan Sideload

Android akan minta izin "Install dari sumber tidak dikenal". Ini normal — tekan Izinkan.

4

Selesai!

Aplikasi WibuOtaku siap dipakai. Buka dari layar beranda atau laci aplikasi.

⚠️ APK ini tidak melalui Google Play Store, jadi Android meminta izin sideload. Ini normal dan aman — seluruh source code terbuka di GitHub untuk kamu audit sendiri.
